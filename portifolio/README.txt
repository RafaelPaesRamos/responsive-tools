Um modelo de portfólio responsivo super simples. É apenas uma única página porque ei, às vezes
você não precisa de meia dúzia de páginas para fazer seu ponto.

Imagens de demonstração* cortesia do Unsplash, uma coleção incrível de imagens CC0 (domínio público)
você pode usar para praticamente qualquer coisa.
(* = Não incluído)

Feedback, relatórios de bugs e comentários não são apenas bem-vindos, mas fortemente encorajados :)

PS: Não sabe como fazer esse formulário de contato funcionar? Experimente o Formspree.io (é incrível).
